注意: Firefox, Safari,和 Chrome 兼容 resize 属性.

调整属性指定一个元素是否由用户可调整大小的。

box-sizing 属性

定义如何计算一个元素的总宽度和总高度，是否包含内边距和边框。

box-sizing: content-box (默认):

高度和宽度只应用于元素的内容:

这个 div 的宽度为 300px。但完整宽度为 300px + 20px (左边框和右边框) + 60px (左边距和右边距) = 380px!

box-sizing: border-box:

高度和宽度应用于元素的所有部分: 内容、内边距和边框:

不管如何这里的完整宽度为300px!

注意: Internet Explorer 不兼容 outline-offset属性.

这个 div有一个轮廓边界15 px边境外的边缘。
